## 📱 STEP 1: 브라우저에서 앱 열기

2. 주소창에 입력: `http://localhost:3001`

## 🗺️ STEP 2: GPS 시뮬레이션 설정 (필수!)

### Chrome DevTools Sensors 사용

1. **F12** 키 눌러 개발자 도구 열기
2. Console 탭 옆 **⋮ (메뉴)** 클릭
3. **More tools** → **Sensors** 선택
4. **Location** 섹션에서:
   - **"Custom location"** 선택
   - Latitude: `37.6524` 입력
   - Longitude: `127.6874` 입력
   - (비발디파크 컨벤션센터 좌표)

5. 헤더에서 **"✓ GPS 활성화"** 표시 확인

### GPS 설정 스크린샷
```
┌─────────────────────────────────┐
│ Location                        │
│ ○ No override                   │
│ ● Custom location               │
│   Latitude:  37.6524            │
│   Longitude: 127.6874           │
│   Timezone: Asia/Seoul          │
└─────────────────────────────────┘
```

## 🎮 STEP 3: 앱 사용 테스트

### 3-1. 홈 화면
- "탐험 시작하기" 버튼 클릭
- → 지도 화면으로 이동

### 3-2. 지도 화면
- **파란 점** = 내 위치 (GPS 기반)
- **9개의 작품 핀** 확인:
  - 🐰 Alice Rabbit (실제 QR)
  - 😸 Cheshire Cat (가상 - SOON 배지)
  - ♥️ Queen of Hearts (가상)
  - 🎩 Mad Hatter (가상)
  - ⏰ White Rabbit (가상)
  - ♠️ Card Soldier (가상)
  - 🐛 Caterpillar (가상)
  - 🦤 Dodo Bird (가상)
  - 🐢 Mock Turtle (가상)

- **각 핀 아래 거리 표시** 확인 (예: 50m, 150m)

### 3-3. 작품 선택
- 🐰 **Alice Rabbit** 핀 클릭
- 하단 시트에서 작품 정보 확인
- GPS 범위 내라면: **"QR 스캔하기"** 버튼 활성화 ✅
- GPS 범위 밖이라면: **"🔒 100m 이내로 이동"** 표시

### 3-4-A. 실제 QR 스캔 (Alice Rabbit)

**준비물:** QR 이미지 파일 `3Dobj_qrcode__2_.png`

1. QR 이미지를 **다른 화면/폰/태블릿**에 띄워놓기
2. "QR 스캔하기" 버튼 클릭
3. **카메라 권한 허용**
4. QR 코드에 카메라 조준
5. ✅ 인식 성공 시:
   - "🎉 실제 작품 수집!" 토스트 메시지
   - SwiftXR AR 링크: https://seoyoung.swiftxr.site/seo-001
   - 컬렉션에 자동 추가

### 3-4-B. 가상 작품 체험 (다른 작품들)

1. 😸 **Cheshire Cat** 등 가상 작품 핀 클릭
2. **"가상 체험하기"** 버튼 클릭 (GPS 불필요)
3. 3.5초 애니메이션 진행
4. ✨ "가상 체험 완료" 메시지
5. 컬렉션에 자동 추가

### 3-5. Collection 확인
- 상단 📦 아이콘 클릭
- 수집한 작품들 그리드로 표시
- 수집 날짜 확인

### 3-6. My Room (가상 갤러리)
- 상단 🏠 아이콘 클릭
- **가상 갤러리** 탭: 드래그 앤 드롭으로 작품 배치
- **실제 방 AR** 탭: AR 시뮬레이션 (바닥 감지 → 작품 배치)

## 🐛 문제 해결

### GPS가 인식 안 돼요
- F12 → Sensors 탭이 열려있는지 확인
- "Custom location" 선택했는지 확인
- 좌표 형식: `37.6524` (쉼표 없음)
- 페이지 새로고침 (F5)

### 카메라가 안 켜져요
- 브라우저 주소창 왼쪽 🔒 아이콘 클릭
- "카메라" 권한 → 허용
- 페이지 새로고침

### QR 코드가 인식 안 돼요
- QR 이미지를 **밝은 화면**에 크게 띄우기
- 카메라와 QR 사이 거리 10-30cm
- 카메라 초점 맞을 때까지 기다리기

### 포트가 막혔어요
```bash
# 3000번 포트 프로세스 종료
killall node

# 다시 시작
npm run dev
```

## 💡 추가 팁

### 모바일 실제 테스트

```bash
# 같은 WiFi에 연결된 모바일에서 테스트
npm run dev

# 터미널에 표시되는 Network 주소 확인
# 예: http://192.168.x.x:3001

# 모바일 브라우저에서 접속
# → 실제 GPS와 카메라 사용 가능!
```

### ngrok으로 외부 접속

```bash
# ngrok 설치 (없으면)
npm install -g ngrok

# 터널 생성
npx ngrok http 3001

# 생성된 https URL로 어디서든 접속 가능
# 예: https://abc123.ngrok.io
```
